# Propagação de saldo credor entre faturas

Quando os estornos de uma fatura superam suas compras, o saldo credor excedente é aplicado explicitamente às faturas seguintes do mesmo cartão de crédito, atravessando quantos ciclos forem necessários até ser consumido. O crédito é vinculado entre faturas com seu valor aplicado, em vez de alterar compras ou criar transações fictícias, para preservar a origem e a auditoria do saldo.
